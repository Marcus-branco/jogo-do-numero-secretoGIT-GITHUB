# Jogo do Número Secreto ALURA: Utilizando ferramentas de versionamento

Este repositório foi criado com o objetivo de consolidar e praticar os conhecimentos adquiridos nos cursos de desenvolvimento de aplicações e sites da **Alura**. Ele funciona como o ponto de partida do meu portfólio, reunindo projetos práticos que demonstram a evolução das minhas habilidades técnicas.

## Objetivos do Repositório

* Aplicar na prática as ferramentas, conceitos e boas práticas apresentadas nos cursos.
*  Documentar minha evolução no desenvolvimento de software para futuras oportunidades profissionais.
*  Adotar padrões de versionamento claros e código limpo desde os primeiros projetos.
